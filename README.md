# Tender Information Extraction System (招标信息提取系统)

## 简介
这是一个自动化的通用招标信息获取系统，旨在从公开网站、需登录平台及微信公众号等多种渠道采集数据，利用 LLM (Gemini) 进行非结构化数据的精准提取，并通过语义分析实现智能推送。
